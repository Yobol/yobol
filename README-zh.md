# yobol

## 前置依赖

### 前端

#### [Vue.js](https://vuejs.org/)

易学易用，性能出色，灵活多变的 Web 前端框架。

- 易学易用: 基于标准 HTML、CSS 和 JavaScript 构建，提供直观的 API 和一流的文档。

- 性能出色：经过编译器优化、完全响应式的渲染系统，几乎不需要手动优化。

- 灵活多变: 丰富的、可渐进式集成的生态系统，可以根据应用规模在库和框架之间切换自如。
