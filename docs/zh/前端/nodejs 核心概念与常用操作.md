# Node.js 核心概念与常用操作

## 依赖管理

### package.json

`package.json` 文件列出了项目所依赖的库和插件。在指定[语义化版本](https://semver.org/)时，支持使用 `^` 和 `~` 符号作为版本号前缀：

- `^` 表示使用 `MAJOR version` 内的最新版本，如 `^1.2.0` 会使用 `2.x.x` 中的最新版本 `2.4.8`；

- `~` 表示使用 `MINOR version` 内的最新版本，如 `~1.2.0` 会使用 `1.2.x` 中的最新版本 `1.2.4`；

- 当不使用上述符号时，表示使用指定版本。
